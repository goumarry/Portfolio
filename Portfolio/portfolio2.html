SAE - 1.04

Création d'une base de données.

Objectif

Création d'une base de données contenant diverses informations sur le Titanic, en binôme.

Compétences

  • Gestion et interrogation de bases de données relationnelles.
  • Travail d'équipe
  • Conception de bases de données relationnelles à partir d'un cahier des charges.

Description

Cette SAE consistait donc d'analyser les données disponibles dans une base de données préexistante contenant des informations sur le Titanic. Il falait d'extraire des informations statistiques pertinentes et de répondre aux questions posées en utilisant les fonctionnalités de requêtes de la base de données Titanic ou de l'application associée.
